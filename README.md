## 💻 설치 방법
1. 저장소 복제
```bash
git clone [레포지토리 URL]
cd [프로젝트명]
```

2. 의존성 설치
```bash
npm install
# 또는
yarn install
```

3. 개발 서버 실행
```bash
npm run dev
# 또는
yarn dev
```

4. 빌드
```bash
npm run build
# 또는
yarn build
```
